# 🌱 Smart Volunteer Management System

A modern Volunteer Registration and Management System developed for the NayePankh Foundation Internship Selection Task.

## 📌 Project Overview

This web application helps manage volunteer registrations efficiently. Volunteers can register through a user-friendly form, and their information is stored locally using Local Storage. An admin dashboard allows viewing, searching, and managing volunteer records.

---

## ✨ Features

### Volunteer Registration
- Register volunteers through an interactive form
- User-friendly and responsive design
- Input validation for required fields

### Duplicate Email Validation
- Prevents multiple registrations using the same email address

### Local Storage Database
- Stores volunteer data in browser Local Storage
- Data remains available after page refresh

### Volunteer Dashboard
- Displays all registered volunteers
- Shows total volunteer count

### Search Functionality
- Search volunteers by:
  - Name
  - Email
  - Department

### Delete Functionality
- Remove volunteer records from the dashboard

### Responsive Design
- Works on desktop, tablet, and mobile devices

## 🚀 How to Run

1. Download the project files
2. Open the project folder
3. Open `index.html` in a web browser
4. Register volunteers
5. Open `dashboard.html` to manage volunteer records
